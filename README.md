Cached-LaTeX-Plugin
===================

A cached LaTeX plugin for Atlassian Confluence. Uses [JLaTeXMath][j] to generate
LaTeX images and caches them as page attachments.

[j]: http://forge.scilab.org/index.php/p/jlatexmath/
